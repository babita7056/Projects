# Pet Care & Appointment Booking Website

A responsive Pet Care Website built using **React + Vite**, converted from a traditional **HTML**, **CSS**, and **JavaScript** project.
This project demonstrates modern React concepts such as components, hooks, state management, routing, and responsive UI design.

## Features
- Pet services overview (Grooming, Veterinary, Training, Boarding, Emergency)
- Appointment booking form
- View booked appointments
- Appointment data stored using localStorage
- Fully responsive design (mobile & desktop)
- Sign-in popup modal
- Single Page Application (SPA) using React

## Technologies Used
- React (Functional Components)
- Vite (Fast build tool)
- JavaScript (ES6)
- HTML5
- CSS3
- React Hooks (useState, useEffect)
- LocalStorage
- Font Awesome Icons

## How It Works
- Appointment Booking
- User fills the appointment form
- Data is saved in browser localStorage
- Clicking "View Appointments" displays stored appointments
- Responsive Navigation
- Desktop: Normal navbar
